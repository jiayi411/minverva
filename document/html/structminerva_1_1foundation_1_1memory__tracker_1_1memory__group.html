Detailed Description

group organized by line, function, size and count

We organize the memory blocks by a key (function) and it's values. Due to a memory allocation may be occured more than once in a single function, so we use a container to distinguish different allocations, besides, the size may vary even in a same line, for instance, array allocation with different size, so we need to handle this too
